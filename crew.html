참여 규칙
①참여 시간 10:00~22:00 / 완료 시간 22:00~다음날 09:00
②해시태그 복사 후 피드에 작성 → 앱에서 참여 등록
③완료 시간에 해시태그 2~3회 재검색, 꼼꼼히 좋아요 → 완료 체크

⑤주 2회 이상 완료 필수 (월~일 기준)
봉사 규칙
①주 완료 0회 또는 1회 → 봉사 1회 (매주 월요일 자동 부여)
②완료 미체크 → 봉사 1회 자동 부여 (매일 09:01)
③봉사 처리: 참여 유형 [봉사 참여] 선택 → 참여+완료 체크
④봉사 발생 후 3일 이내 미처리 → 계정 자동 잠금
⑤봉사 3회 누적 → 계정 즉시 잠금
